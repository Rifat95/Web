indentation: tabs
continuous indentation: 1 tab
if/class/method line break: 2 tabs
brace placement: same line
html/css: file-case, simple quotes
less variables: snake_case
javascript: camelCase, simple quotes
sql tables: PascalCase
sql fields: camelCase

javadoc:
- block comments: full punctuation
- single line: first letter in uppercase, without punctuation
- @tags: lowercase, without punctuation

class members order:
- static fields
- normal fields
- constructor
- static methods
- normal methods (sorted by visibility)
